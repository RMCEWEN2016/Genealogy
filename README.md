How to read the chart:
The lines labelled 1-18 is a one sided tree with the line of ancestry leading up to me.
Past the 18 labelled lines is the trees of the spouses of whoever is listed on the 
corresponding line number. On that line the tree starts back as far as known and leads
up to the spouse from the original tree at the very end. These separate trees also show
siblings and children not listed on the original tree, with numbers indented after the
parents. If at any point in the tree there is someone who is also married and there is
additional information on the spouses parents, the tree will break again and have a
line number and sub number (like 17-1-2) which starts another break in the tree where it
starts down the tree as far as known and works down to the spouse. If there is a break-
off tree the tree will NOT go past the spouses name, as you can refer to the number it
is under to find the lineage past them.
